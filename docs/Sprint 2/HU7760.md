### Descripción
Como usuario, quiero visualizar los Objetivos de Programa (OPP) para poder entender los objetivos que se han establecido para los diferentes programas académicos.

### Criterios de Aceptación
- Se debe crear un endpoint GET `/program-objectives` que permita listar todos los Objetivos de Programa disponibles.
- La respuesta del endpoint debe incluir una lista de OPPs con sus respectivos identificadores y descripciones.
- El endpoint debe manejar adecuadamente los errores, devolviendo un mensaje claro en caso de que no se encuentren OPPs.
