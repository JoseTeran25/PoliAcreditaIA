# HU8073 — Listar Resultados de Aprendizaje de Asignatura (RAA)

## Descripción
Como usuario, quiero poder listar los Resultados de Aprendizaje Asociados (RAA) a una asignatura para entender mejor los objetivos de aprendizaje de cada curso.

## Criterios de Aceptación
1. El sistema debe proporcionar un endpoint que permita obtener los RAAs asociados a una asignatura específica.
2. La consulta debe ser realizada utilizando el ID de la asignatura.
3. La respuesta debe incluir todos los RAAs relacionados con la asignatura de manera clara y estructurada.
4. El sistema debe manejar correctamente los casos en los que no existan RAAs asociados a la asignatura.

## Tareas (Backend)
- [ ] Crear un endpoint API `GET /api/subjects/{id}/learning-outcomes` para obtener los RAAs asociados a una asignatura (Tarea ID: 8123).
- [ ] Implementar la lógica de consulta de RAAs por ID de asignatura, asegurando que se devuelvan los datos correctos (Tarea ID: 8124).
- [ ] Implementar pruebas unitarias para asegurar que el endpoint funciona correctamente y devuelve los resultados esperados.
- [ ] Documentar el nuevo endpoint y su uso en la API.

## Tareas (Frontend)
- [ ] Crear una interfaz que permita al usuario ver los RAAs de una asignatura al seleccionar una asignatura específica.
- [ ] Implementar la lógica para enviar la solicitud al backend y manejar la respuesta con los RAAs.
- [ ] Mostrar los RAAs en un formato claro y accesible para el usuario.
- [ ] Manejar el caso en que no se encuentren RAAs y mostrar un mensaje adecuado al usuario.
