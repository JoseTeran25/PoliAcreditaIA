# HU8071 — Paginar Asignaturas (PEA)

## Descripción
Como usuario, quiero poder paginar la lista de asignaturas para facilitar la navegación y mejorar la experiencia de usuario al visualizar grandes volúmenes de datos.

## Criterios de Aceptación
1. El sistema debe permitir la paginación de los resultados en el endpoint de listado de asignaturas.
2. El usuario debe poder especificar el número de resultados por página y la página que desea visualizar.
3. La respuesta del servidor debe incluir información sobre el total de páginas y el número total de asignaturas.
4. La paginación debe ser eficiente y no afectar el rendimiento del sistema.

## Tareas (Backend)
- [ ] Extender la lógica de paginación en el endpoint de listado de asignaturas para manejar los parámetros de paginación (Tarea ID: 8104).
- [ ] Optimizar la consulta de base de datos para mejorar la eficiencia de la paginación (Tarea ID: 8105).
- [ ] Implementar pruebas unitarias para asegurar que la paginación funciona correctamente y devuelve los resultados esperados.
- [ ] Documentar el nuevo comportamiento del endpoint y sus parámetros de paginación en la API.

## Tareas (Frontend)
- [ ] Crear controles de paginación en la interfaz de usuario para que los usuarios puedan navegar entre las páginas de resultados.
- [ ] Implementar la lógica para enviar los parámetros de paginación al backend y manejar la respuesta.
- [ ] Mostrar el número total de asignaturas y la información de paginación en la interfaz de usuario.
- [ ] Asegurar que la interfaz se actualice correctamente al cambiar de página.
