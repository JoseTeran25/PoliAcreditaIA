## Descripción
Como coordinador o profesor, quiero poder crear una nueva asignatura en el sistema, de manera que se pueda gestionar adecuadamente la oferta académica.

## Criterios de Aceptación
1. El sistema debe permitir la creación de una nueva asignatura mediante un formulario que incluya todos los campos necesarios.
2. La validación debe asegurarse de que todos los datos requeridos sean correctos y estén completos antes de guardar la asignatura.
3. La nueva asignatura debe ser persistida en la base de datos y estar disponible para su consulta posterior.
4. El sistema debe devolver una respuesta de éxito o error al intentar crear la asignatura.
