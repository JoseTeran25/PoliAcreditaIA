# HU7877 — Registrar foto al usuario

## Descripción
Como usuario, quiero poder registrar una foto en mi perfil para personalizar mi experiencia en la aplicación.

## Criterios de Aceptación
1. El sistema debe permitir al usuario subir una imagen desde su dispositivo.
2. La imagen debe ser validada para asegurar que cumple con los formatos permitidos (JPEG, PNG).
3. El tamaño máximo de la imagen no debe exceder 5MB.
4. Una vez subida, la imagen debe ser almacenada en el servidor y asociada al perfil del usuario.
5. El usuario debe recibir una notificación de éxito o error tras intentar subir la imagen.

## Tareas (Backend)
- [ ] Implementar un endpoint `POST /api/users/{id}/photo` para manejar la subida de la foto.
- [ ] Validar el formato y tamaño de la imagen en el servidor.
- [ ] Almacenar la imagen en el sistema de archivos o en un servicio de almacenamiento en la nube.
- [ ] Actualizar el modelo de usuario para incluir la URL de la imagen.
- [ ] Implementar pruebas unitarias para asegurar el correcto funcionamiento del endpoint y la validación de la imagen.
- [ ] Documentar el endpoint en la API.

## Tareas (Frontend)
- [ ] Crear un formulario para que el usuario pueda seleccionar y subir su foto.
- [ ] Implementar la lógica para manejar la respuesta del servidor y mostrar mensajes de éxito o error.
- [ ] Asegurar que el formulario valide el tipo y tamaño de la imagen antes de enviarla al servidor.
