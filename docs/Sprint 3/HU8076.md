# HU8076 — Visualizar Matriz de Contribución RAA vs RA

## Descripción
Como usuario, quiero poder visualizar la matriz de contribución entre los Resultados de Aprendizaje Asociados (RAA) y los Resultados de Aprendizaje (RA) para entender cómo se alinean los objetivos de aprendizaje con los resultados esperados de la asignatura.

## Criterios de Aceptación
1. El sistema debe generar una matriz que muestre la relación entre los RAAs y los RAs de una asignatura específica.
2. La matriz debe ser accesible a través de un endpoint API que permita consultar la información por ID de asignatura.
3. La visualización debe ser clara y permitir al usuario identificar fácilmente las contribuciones de cada RAA a los RAs.
4. El sistema debe manejar correctamente los casos en los que no existan RAAs o RAs asociados a la asignatura.

## Tareas (Backend)
- [ ] Implementar la lógica para generar la matriz de contribución entre RAAs y RAs (Tarea ID: 8141).
- [ ] Crear un endpoint API `GET /api/subjects/{id}/contribution-matrix` para consultar la matriz RAA vs RA por asignatura (Tarea ID: 8146).
- [ ] Implementar pruebas unitarias para asegurar que la generación de la matriz funciona correctamente y devuelve los resultados esperados.
- [ ] Documentar el nuevo endpoint y su uso en la API.

## Tareas (Frontend)
- [ ] Crear una interfaz que permita al usuario seleccionar una asignatura para visualizar la matriz de contribución.
- [ ] Implementar la lógica para enviar la solicitud al backend y manejar la respuesta con la matriz de contribución.
- [ ] Mostrar la matriz en un formato visual claro y accesible para el usuario.
- [ ] Manejar el caso en que no se encuentren RAAs o RAs y mostrar un mensaje adecuado al usuario.
