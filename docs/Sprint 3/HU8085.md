# HU8085 — Filtrar Asignaturas por Periodo académico (PEA)

## Descripción
Como usuario, quiero poder filtrar las asignaturas disponibles por periodo académico, de manera que pueda encontrar fácilmente aquellas que se ofrecen en un periodo específico.

## Criterios de Aceptación
1. El sistema debe permitir filtrar el listado de asignaturas por periodo académico, mostrando solo aquellas que corresponden al periodo seleccionado.
2. El filtro de periodo académico debe ser combinable con otros criterios de búsqueda, como nombre o código de la asignatura.
3. La respuesta del sistema debe incluir solo las asignaturas que cumplen con los criterios de filtrado aplicados.
4. El sistema debe manejar correctamente los casos en los que no existan asignaturas para el periodo académico seleccionado.

## Tareas (Backend)
- [ ] Extender la lógica de filtrado por periodo académico en el endpoint de listado de asignaturas para permitir la consulta basada en este criterio (Tarea ID: 8101).
- [ ] Asegurar que el filtro de periodo académico sea combinable con otras búsquedas, permitiendo una búsqueda más precisa (Tarea ID: 8102).
- [ ] Implementar pruebas unitarias para asegurar que el filtrado por periodo académico funciona correctamente y que se respetan los criterios de búsqueda combinados.
- [ ] Documentar el nuevo comportamiento del endpoint y sus parámetros de filtrado en la API.

## Tareas (Frontend)
- [ ] Crear un campo de selección en la interfaz de usuario que permita al usuario elegir un periodo académico para filtrar las asignaturas.
- [ ] Modificar la lógica para enviar solicitudes al backend que incluyan el filtro de periodo académico junto con otros criterios de búsqueda.
- [ ] Mostrar el listado de asignaturas filtradas en la interfaz de usuario, asegurando que se reflejen los criterios seleccionados.
- [ ] Manejar el caso en que no se encuentren asignaturas para el periodo académico seleccionado y mostrar un mensaje adecuado al usuario.
