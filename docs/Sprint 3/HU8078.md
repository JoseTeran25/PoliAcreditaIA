# HU8078 — Listar Resultados de Aprendizaje (RA) para selección

## Descripción
Como usuario, quiero poder listar los Resultados de Aprendizaje (RA) disponibles para una carrera específica, de manera que pueda seleccionar los que se alineen con los Resultados de Aprendizaje Asociados (RAA) de una asignatura.

## Criterios de Aceptación
1. El sistema debe permitir la consulta de RAs filtrados por carrera.
2. La respuesta debe incluir un listado claro de los RAs disponibles para la carrera seleccionada.
3. El sistema debe manejar correctamente los casos en los que no existan RAs asociados a la carrera consultada.
4. La interfaz debe permitir al usuario seleccionar uno o más RAs de la lista para su posterior uso.

## Tareas (Backend)
- [ ] Implementar la lógica de consulta de RAs por carrera, asegurando que se devuelvan los datos correctos (Tarea ID: 8152).
- [ ] Crear un endpoint API `GET /api/careers/{id}/learning-outcomes` para obtener el listado de RAs asociados a una carrera (Tarea ID: 8153).
- [ ] Implementar pruebas unitarias para asegurar que la consulta de RAs funciona correctamente y devuelve los resultados esperados.
- [ ] Documentar el nuevo endpoint y su uso en la API.

## Tareas (Frontend)
- [ ] Crear una interfaz que permita al usuario seleccionar una carrera para visualizar los RAs disponibles.
- [ ] Implementar la lógica para enviar la solicitud al backend y manejar la respuesta con el listado de RAs.
- [ ] Mostrar los RAs en un formato claro y accesible, permitiendo al usuario seleccionar los que desee.
- [ ] Manejar el caso en que no se encuentren RAs y mostrar un mensaje adecuado al usuario.
