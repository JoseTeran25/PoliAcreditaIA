## Descripción
Como usuario, quiero poder crear relaciones de mapeo entre Resultados de Aprendizaje (RA) y Criterios EUR-ACE en lote, para facilitar la gestión y justificación de estas relaciones en el sistema.

## Criterios de Aceptación
1. Se debe crear un endpoint POST en la ruta `/mappings/ra-eur-ace/batch` que permita guardar múltiples relaciones de mapeo entre RAs y Criterios EUR-ACE.
2. La solicitud debe incluir un cuerpo en formato JSON que contenga un array de objetos, donde cada objeto represente una relación de mapeo y contenga al menos los siguientes campos:

   - `eurAceId`: el identificador del Criterio EUR-ACE.
   - `justificacion`: una justificación para la relación de mapeo.
3. El backend debe implementar validaciones para evitar duplicados en las relaciones de mapeo. Si se intenta crear una relación que ya existe, el endpoint debe devolver un código de estado HTTP 400 y un mensaje de error claro.
