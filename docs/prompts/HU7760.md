## Descripción
Como usuario, quiero poder visualizar los Objetivos de Programa (OPP) a través de un endpoint, para acceder fácilmente a la información sobre los objetivos establecidos para los programas educativos.

2. La respuesta del endpoint debe ser un JSON que contenga una lista de objetos, donde cada objeto representa un Objetivo de Programa, incluyendo al menos los siguientes campos:

   - nombre
   - descripción
3. Si no se encuentran Objetivos de Programa, el endpoint debe devolver un array vacío.
