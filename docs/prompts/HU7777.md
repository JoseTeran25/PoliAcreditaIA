# HU7777 - Visualizar Matriz de Mapeo OPP vs RA

## Descripción
Como usuario, quiero poder visualizar la matriz de mapeo entre Objetivos de Programa (OPP) y Resultados de Aprendizaje (RA), para entender mejor las relaciones y justificaciones entre ellos.

## Criterios de Aceptación
1. Se debe crear un endpoint GET en la ruta `/mappings/opp-ra` que permita obtener los datos de la matriz de relaciones entre OPP y RA.
2. La respuesta del endpoint debe incluir un array de objetos que representen las relaciones de mapeo, donde cada objeto contenga al menos los siguientes campos:
   - `oppId`: el identificador del Objetivo de Programa.

   - `justificacion`: la justificación para la relación de mapeo.
3. Se debe exponer un catálogo de OPP por programa a través del endpoint GET `/programs/{programId}/opps?lang=es`, que devuelva un array de OPPs con los siguientes campos:

   - `code`: código del OPP.
   - `name`: nombre del OPP.
   - `description`: descripción del OPP.
   - `active`: estado activo del OPP.
4. Se debe exponer un catálogo de RA por programa a través del endpoint GET `/programs/{programId}/learning-outcomes?lang=es`, que devuelva un array de RAs con los siguientes campos:

   - `code`: código del RA.
   - `name`: nombre del RA.
   - `description`: descripción del RA.
   - `active`: estado activo del RA.
5. Se debe manejar adecuadamente cualquier error que pueda ocurrir durante la ejecución de los endpoints, devolviendo un mensaje de error claro y un código de estado HTTP apropiado.
