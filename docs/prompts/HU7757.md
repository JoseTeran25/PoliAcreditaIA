# HU7757 - Visualizar Criterios EUR-ACE

## Descripción
Como usuario, quiero poder visualizar los criterios EUR-ACE a través de un endpoint, para acceder fácilmente a la información sobre estos criterios de calidad en la educación.

## Criterios de Aceptación
1. Se debe crear un endpoint GET en la ruta `/eur-ace-criteria` que permita listar todos los criterios EUR-ACE.
2. La respuesta del endpoint debe ser un JSON que contenga una lista de objetos, donde cada objeto representa un criterio EUR-ACE, incluyendo al menos los siguientes campos:
   - id
   - nombre
   - descripción
3. Si no se encuentran criterios EUR-ACE, el endpoint debe devolver un array vacío.
4. Se debe manejar adecuadamente cualquier error que pueda ocurrir durante la ejecución del endpoint, devolviendo un mensaje de error claro y un código de estado HTTP apropiado.
