# HU8074 — Crear Resultado de Aprendizaje de Asignatura (RAA)

## Descripción
Como usuario, quiero poder crear un Resultado de Aprendizaje Asociado (RAA) para una asignatura, de manera que se puedan establecer claramente los objetivos de aprendizaje del curso.

## Criterios de Aceptación
1. El sistema debe permitir la creación de un RAA asociado a una asignatura mediante un formulario.
2. Los campos requeridos para el RAA deben incluir Código, Tipo y Descripción.
3. La validación debe asegurarse de que todos los campos sean correctos y estén completos antes de guardar el RAA.
4. El sistema debe devolver una respuesta de éxito o error al intentar crear un RAA.
5. El nuevo RAA debe ser visible en la lista de RAAs de la asignatura correspondiente tras su creación.

## Tareas (Backend)
- [ ] Implementar la lógica de validación y guardado de RAA para una asignatura (Tarea ID: 8125).
- [ ] Crear un endpoint API `POST /api/subjects/{id}/learning-outcomes` para crear un nuevo RAA (Tarea ID: 8128).
- [ ] Definir el modelo de datos para RAA, que incluya los campos Código, Tipo y Descripción (Tarea ID: 8135).
- [ ] Implementar pruebas unitarias para asegurar que la creación de RAAs funciona correctamente y que se manejan los errores adecuadamente.
- [ ] Documentar el nuevo endpoint y su uso en la API.

## Tareas (Frontend)
- [ ] Crear un formulario en la interfaz de usuario para permitir la entrada de los datos del nuevo RAA.
- [ ] Implementar la lógica para enviar los datos del formulario al backend y manejar la respuesta.
- [ ] Mostrar un mensaje de éxito o error al usuario tras intentar crear el RAA.
- [ ] Actualizar la lista de RAAs de la asignatura para incluir el nuevo RAA tras su creación.
