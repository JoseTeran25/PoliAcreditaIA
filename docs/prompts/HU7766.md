## Descripción
Como usuario, quiero poder buscar Objetivos de Programa (OPP) utilizando filtros por código y descripción, para facilitar la localización de información específica en la lista de OPPs.

1. Se debe adaptar el endpoint existente para listar Objetivos de Programa (OPP) para que acepte parámetros de consulta opcionales `codigo` y `descripcion`.

3. Si se proporciona el parámetro `descripcion`, el endpoint debe filtrar los resultados para incluir solo aquellos OPPs cuya descripción contenga el valor proporcionado.
4. La respuesta del endpoint debe ser un JSON que contenga una lista de objetos, donde cada objeto representa un Objetivo de Programa, incluyendo al menos los siguientes campos:
   - id
   - codigo
   - nombre
   - descripcion
5. Si no se encuentran OPPs que coincidan con los filtros aplicados, el endpoint debe devolver un array vacío.
6. Se debe manejar adecuadamente cualquier error que pueda ocurrir durante la ejecución del endpoint, devolviendo un mensaje de error claro y un código de estado HTTP apropiado.
