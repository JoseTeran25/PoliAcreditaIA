# HU7751 - Visualizar Resultados de Aprendizaje Específicos (RE)

## Descripción
Como usuario, quiero poder visualizar los Resultados de Aprendizaje (RAs) específicos a través de un endpoint, para poder filtrar y obtener información relevante sobre los mismos.

## Criterios de Aceptación
1. Se debe crear un endpoint GET en la ruta `/learning-outcomes` que permita listar los Resultados de Aprendizaje.
2. El endpoint debe aceptar un parámetro de consulta `tipo` que, cuando se establece en 'Específico', filtrará los resultados para mostrar únicamente los RAs de este tipo.
3. La respuesta del endpoint debe ser un JSON que contenga una lista de objetos, donde cada objeto representa un Resultado de Aprendizaje específico, incluyendo al menos los siguientes campos:
   - id
   - nombre
   - descripción
   - tipo
4. Si no se encuentran RAs específicos, el endpoint debe devolver un array vacío.
5. Se debe manejar adecuadamente cualquier error que pueda ocurrir durante la ejecución del endpoint, devolviendo un mensaje de error claro y un código de estado HTTP apropiado.

## Tareas
- [7828] [BE] Crear/adaptar endpoint GET /learning-outcomes para listar RAs con filtro por tipo='Específico'
