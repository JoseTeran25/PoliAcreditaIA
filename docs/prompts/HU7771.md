## HU7771 - Crear Relación de Mapeo OPP vs RA (Asistente)

### Descripción
Como usuario, quiero crear relaciones de mapeo entre Objetivos de Programa (OPP) y Resultados de Aprendizaje (RA) en lote, para facilitar la gestión y justificación de estas relaciones en el sistema.

### Criterios de Aceptación
- Se debe crear un endpoint POST `/mappings/opp-ra/batch` que permita guardar múltiples relaciones y justificaciones en una sola solicitud.
- El cuerpo de la solicitud debe incluir un array de objetos que contengan las relaciones entre OPPs y RAs, junto con sus justificaciones.
- Se deben implementar validaciones para la creación de mapeos en lote, como la verificación de duplicados, asegurando que no se creen relaciones repetidas.
- Se deben adaptar los endpoints de listar OPPs y RAs para que sean utilizables en el asistente, incluyendo funcionalidades de búsqueda, paginación y filtrado.
