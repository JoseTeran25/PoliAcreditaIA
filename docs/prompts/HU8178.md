```markdown
# HU8178 — Paginar Resultados de Aprendizaje (RAA) para selección

## Descripción
Como usuario, quiero poder paginar los Resultados de Aprendizaje de Asignatura (RAA) en el listado, de manera que pueda navegar fácilmente a través de grandes conjuntos de datos.

## Criterios de Aceptación
1. El sistema debe permitir la paginación de los RAAs en el listado, mostrando un número limitado de resultados por página.
2. La respuesta del sistema debe incluir información sobre la página actual, el total de páginas y el número total de RAAs disponibles.
3. El sistema debe manejar correctamente los casos en los que no existan RAAs disponibles, así como la navegación a páginas que no contienen resultados.

## Tareas (Backend)
- [ ] Extender la lógica de paginación en el endpoint de listado de RAAs para permitir la consulta paginada (Tarea ID: 8181).
- [ ] Implementar pruebas unitarias para asegurar que la paginación de RAAs funciona correctamente y que se devuelven los resultados esperados.
- [ ] Documentar el nuevo comportamiento del endpoint y sus parámetros de paginación en la API.

## Tareas (Frontend)
- [ ] Crear controles de paginación en la interfaz de usuario que permitan al usuario navegar entre las páginas de resultados.
- [ ] Modificar la lógica para enviar solicitudes al backend que incluyan los parámetros de paginación (número de página y tamaño de página).
- [ ] Mostrar el listado de RAAs paginados en la interfaz de usuario, asegurando que se reflejen los resultados de la página seleccionada.
- [ ] Manejar el caso en que no se encuentren RAAs disponibles y mostrar un mensaje adecuado al usuario.
```
