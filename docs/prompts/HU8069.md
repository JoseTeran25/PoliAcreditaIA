# HU8069 — Buscar Asignaturas (PEA)

## Descripción
Como usuario, quiero poder buscar asignaturas en el sistema para encontrar fácilmente la información que necesito sobre ellas.

## Criterios de Aceptación
1. El sistema debe permitir al usuario realizar búsquedas por nombre de asignatura, código o profesor.
2. La búsqueda debe ser sensible a mayúsculas y minúsculas.
3. Los resultados de la búsqueda deben mostrarse en un formato claro y ordenado.
4. El sistema debe manejar correctamente los casos en los que no se encuentren asignaturas que coincidan con la búsqueda.
5. La búsqueda debe ser eficiente y no afectar el rendimiento del sistema.

## Tareas (Backend)
- [ ] Implementar un endpoint `GET /api/subjects/search` que reciba parámetros de búsqueda.
- [ ] Optimizar la consulta de base de datos para mejorar la velocidad de búsqueda (Tarea ID: 8093).
- [ ] Extender la lógica de búsqueda en el endpoint de listado de asignaturas para incluir los nuevos criterios de búsqueda (Tarea ID: 8094).
- [ ] Implementar pruebas unitarias para asegurar que la búsqueda funciona correctamente y devuelve los resultados esperados.
- [ ] Documentar el nuevo endpoint y sus parámetros en la API.

## Tareas (Frontend)
- [ ] Crear un campo de búsqueda en la interfaz de usuario para que los usuarios puedan ingresar sus criterios de búsqueda.
- [ ] Implementar la lógica para enviar la solicitud de búsqueda al backend y manejar la respuesta.
- [ ] Mostrar los resultados de la búsqueda en un formato amigable y permitir al usuario seleccionar una asignatura para ver más detalles.
- [ ] Manejar el caso en que no se encuentren resultados y mostrar un mensaje adecuado al usuario.
