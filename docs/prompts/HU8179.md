# HU8179 — Buscar Resultados de Aprendizaje (RAA) para selección

## Descripción
Como usuario, quiero poder buscar Resultados de Aprendizaje de Asignatura (RAA) mediante un término de búsqueda, de manera que pueda encontrar rápidamente aquellos que son relevantes para mis necesidades académicas.

## Criterios de Aceptación
1. El sistema debe permitir realizar búsquedas de RAAs utilizando un término de búsqueda que puede incluir parte del nombre o código del RAA.
2. La respuesta del sistema debe incluir solo los RAAs que coincidan con el término de búsqueda proporcionado.
3. El sistema debe manejar correctamente los casos en los que no se encuentren RAAs que coincidan con el término de búsqueda.

## Tareas (Backend)
- [ ] Extender la lógica de búsqueda en el endpoint de listado de RAAs para permitir la consulta basada en un término de búsqueda (Tarea ID: 8185).
- [ ] Implementar pruebas unitarias para asegurar que la búsqueda de RAAs funciona correctamente y devuelve los resultados esperados.
- [ ] Documentar el nuevo comportamiento del endpoint y sus parámetros de búsqueda en la API.

## Tareas (Frontend)
- [ ] Crear un campo de entrada en la interfaz de usuario que permita al usuario ingresar un término de búsqueda para los RAAs.
- [ ] Modificar la lógica para enviar solicitudes al backend que incluyan el término de búsqueda.
- [ ] Mostrar el listado de RAAs que coincidan con el término de búsqueda en la interfaz de usuario.
- [ ] Manejar el caso en que no se encuentren RAAs que coincidan con el término de búsqueda y mostrar un mensaje adecuado al usuario.
