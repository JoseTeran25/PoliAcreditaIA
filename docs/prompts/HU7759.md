# HU7759 - Buscar Resultados de Aprendizaje (RA)

## Descripción
Como usuario, quiero poder buscar Resultados de Aprendizaje (RA) utilizando filtros por código y descripción, para facilitar la localización de información específica en la lista de RAs.

## Criterios de Aceptación
1. Se debe adaptar el endpoint existente para listar Resultados de Aprendizaje (RA) para que acepte parámetros de consulta opcionales `codigo` y `descripcion`.
2. Si se proporciona el parámetro `codigo`, el endpoint debe filtrar los resultados para incluir solo aquellos RAs cuyo código coincida con el valor proporcionado.
3. Si se proporciona el parámetro `descripcion`, el endpoint debe filtrar los resultados para incluir solo aquellos RAs cuya descripción contenga el valor proporcionado.
4. La respuesta del endpoint debe ser un JSON que contenga una lista de objetos, donde cada objeto representa un Resultado de Aprendizaje, incluyendo al menos los siguientes campos:
   - id
   - codigo
   - nombre
   - descripcion
5. Si no se encuentran RAs que coincidan con los filtros aplicados, el endpoint debe devolver un array vacío.
6. Se debe manejar adecuadamente cualquier error que pueda ocurrir durante la ejecución del endpoint, devolviendo un mensaje de error claro y un código de estado HTTP apropiado.
