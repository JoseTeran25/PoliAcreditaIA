# HU7761 - Crear Objetivo de Programa

## Descripción
Como usuario, quiero poder crear un nuevo Objetivo de Programa (OPP) a través de un endpoint, para poder añadir objetivos que guíen el desarrollo de los programas educativos.

## Criterios de Aceptación
1. Se debe crear un endpoint POST en la ruta `/program-objectives` que permita agregar nuevos Objetivos de Programa (OPP).
2. La solicitud debe incluir un cuerpo en formato JSON que contenga al menos los siguientes campos:
   - `codigo`: un código único para el OPP.
   - `nombre`: el nombre del OPP.
   - `descripcion`: una descripción del OPP.
3. El backend debe validar que el campo `codigo` sea único. Si se intenta agregar un OPP con un código ya existente, el endpoint debe devolver un código de estado HTTP 400 y un mensaje de error claro.
4. Si el OPP se crea correctamente, el endpoint debe devolver un código de estado HTTP 201 y el objeto del OPP creado.
5. Se debe manejar adecuadamente cualquier error que pueda ocurrir durante la ejecución del endpoint, devolviendo un mensaje de error claro y un código de estado HTTP apropiado.

## Tareas
- [7779] [BE] Crear endpoint POST /program-objectives para agregar OPPs
- [7780] [BE] Validar unicidad del código del OPP en backend
